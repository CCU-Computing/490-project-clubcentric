# Finished User Stories 

## Lauren 
- As a student user, I want to be able to edit the information on my profile so that my personal details, such as my name and bio, are always accurate and up to date for other students and club organizers to see. 
- As a user, I want to see the clubs that I am a member/organizer of on my profile, so that I can quickly access the pages of the communities I have joined without searching for them individually. 
- As a student organizer, I want to enter a unique name and description for my club so that potential members understand what my club is about. 
- As a student organizer, I want to easily access a 'Create Club' option directly from the clubs grid so that I can start the process of forming a new organization without searching for a hidden menu. 
- As a student organizer I want to see the existing clubs before creating a club, so that the club that I want to create is not like an existing club. 
- As a system, I want to prevent users from creating clubs with empty names or descriptions so that the platform does not become cluttered with blank or incomplete club entries. 
## Liam

- As a developer, I want to make our website look professional for students and other users so that they can enjoy our website. 
- As a club leader, I want a place to upload documents for club members to read so that they can remain informed on club activities/news.
