# User Stories Backlog 

## Club Page 
- As a student/user, I want to be able to see images for various clubs so that I know what I’m looking for. 
## Profile 
- As a user, I want to see a personalized calendar on my profile page so that I can see the events of the clubs that I am a part of. 
- As a user, I want to be able to hide my email on my profile page so that I can maintain privacy. 
- As a user, I want to upload and display a custom profile picture so that I can personalize my account and be easily recognizable to my - peers. 
- As a user, I want to be prompted to join clubs when I am not a part of any while on my profile page, so that I am more encouraged to get involved with campus organizations. 
- As a student user, I want to display my major on my profile page, so that I can connect with people in similar majors. 
- As a user, I want to be the only person to access the ‘Edit Profile’ button on my profile, so that others cannot accidentally or maliciously change my personal information. 
- As a user, I want to be able to view the profiles of other students so that I can see their major, bio, and the clubs they are associated with. 
## Documents 
- As a club organizer, I want to be able to upload documents to the website so that other users can read them for whatever reason. 
- As a user, I want to be able to find documents related to whatever club(s) I follow so that I can keep up with club activities.
